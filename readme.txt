石川県能登半島地震の画像データ収集
地理院で大体の大画像の範囲を決める
各範囲の分割数と分割画像の4隅座標を計算する　Driveのスプレッドシート
座標打ち込み.ahkで大画像を地理院からDL→Largepngに保存
このとき四隅の座標メモ
画像一覧の縦横サイズもImageSize2CSVで出力　
画像の縦横サイズと四隅の座標からVBA使ってGCPファイル作成
GDALでGCP合わせてGeoTIFFに変換
QGISに読み込んで張り付ける
切り取るためのポリゴンをPolygonGenerationでQGIS上で生成
同一レイヤ上にポリゴンを置いたのでこれを別レイヤに分ける Polygon2MultiLayer
各レイヤでクリップしてデータ作成　PNGOutput

大画像一気にDLしてくるのは無理なので分割してDLしている
また，雲とかがかぶらないように＆四角形で大画像にできるようにarea分けしている
area毎に画像を切り取り　Xがついてるのは能登里山街道以外ということ

areaBはこれ以上東側とか調べようとすると雲がひどくて被害ありかどうかすらそもそもわからん
